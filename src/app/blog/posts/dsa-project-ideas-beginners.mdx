## 2. âœ… Basic Calculator
**DSA Concepts Practiced:**
- Stacks: Evaluate expressions using infix to postfix conversion
- Queues: Process operations in order

**Features:** 
- Perform basic arithmetic (+, -, *, /)
- Handle parentheses and operator precedence

```js
// Example: Using a stack to evaluate postfix expression
function evaluatePostfix(expr) {
  const stack = [];
  for (let token of expr.split(" ")) {
    if (!isNaN(token)) stack.push(Number(token));
    else {
      const b = stack.pop();
      const a = stack.pop();
      stack.push(eval(`${a} ${token} ${b}`));
    }
  }
  return stack.pop();
}
```

This project teaches algorithmic thinking and use of stacks in real applications.
